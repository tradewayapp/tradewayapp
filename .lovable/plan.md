

# Export All Screens as PNG Screenshots

## What
Take screenshots of every screen in the TradeWay app at mobile resolution (390×844) and bundle them into a ZIP file — so your Flutter developer gets visual references instead of code files.

## Screens to Capture
1. Onboarding (all 4 slides)
2. Login
3. OTP
4. Profile Setup
5. Dashboard
6. Wallet
7. Referrals
8. More/Menu
9. KYC
10. Notifications
11. Help Center
12. FAQ
13. Support
14. Privacy Policy
15. Terms
16. Learn
17. AI Guide

## How
- Navigate to each route in the browser at 390×844 viewport
- Take a screenshot of each screen
- Save all PNGs to `/mnt/documents/TradeWay_Screenshots/`
- Bundle into `/mnt/documents/TradeWay_Screenshots.zip`

## Output
A ZIP containing ~20 PNG files named by screen (e.g., `01_Onboarding_Slide1.png`, `05_Dashboard.png`, etc.)
